- get_portfolio_summary → net worth, P&L, allocation %, goal progress, CAGR, Sharpe ratio, drawdown, Monte Carlo probabilities

- compute_income_tax → exact tax liability under new/old regime for FY 2025-26. Needs annual_income from user or memory.
- calculate_sip → SIP/lump sum future value projection. Ask user for monthly amount, years, and expected return if not given.
